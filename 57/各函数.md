
### 一、Socket 生命周期管理函数

这类函数用于创建、配置和销毁Socket。

#### 1. `socket()` - 创建Socket（申请电话机）
**功能**：创建一个Socket端点，相当于申请一部电话机。
```c
#include <sys/socket.h>
int socket(int domain, int type, int protocol);
```
**参数**：
- `domain`：协议族/地址族
  - `AF_UNIX`/**`AF_LOCAL`**：UNIX域Socket（**同一台机器内部通信**）
  - `AF_INET`：IPv4网络通信
  - `AF_INET6`：IPv6网络通信
- `type`：Socket类型
  - **`SOCK_STREAM`**：流式Socket（像打电话，可靠、有序）
  - **`SOCK_DGRAM`**：数据报Socket（像发短信，无连接）
- `protocol`：具体协议，通常填`0`（自动选择）

**返回值**：成功返回Socket文件描述符，失败返回-1。

#### 2. `bind()` - 绑定地址（分配电话号码）
**功能**：将Socket与一个地址绑定，让其他进程能找到它。
```c
#include <sys/socket.h>
int bind(int sockfd, const struct sockaddr *addr, socklen_t addrlen);
```
**参数**：
- `sockfd`：socket()返回的文件描述符
- `addr`：指向地址结构的指针（对UNIX域是`struct sockaddr_un`）
- `addrlen`：地址结构的长度

**UNIX域地址结构**：
```c
#include <sys/un.h>
struct sockaddr_un {
    sa_family_t sun_family;    // 地址族：AF_UNIX
    char        sun_path[108];  // 路径名，如"/tmp/mysocket"
};
```

#### 3. `close()` - 关闭Socket（挂断电话）
**功能**：关闭Socket，释放资源。
```c
#include <unistd.h>
int close(int sockfd);
```
**参数**：
- `sockfd`：要关闭的Socket文件描述符

### 二、面向连接（流式Socket）专用函数

用于`SOCK_STREAM`类型Socket，像打电话需要建立连接。

#### 4. `listen()` - 监听连接（电话待机）
**功能**：将Socket设置为监听模式，准备接受连接请求。
```c
#include <sys/socket.h>
int listen(int sockfd, int backlog);
```
**参数**：
- `sockfd`：已绑定的Socket描述符
- `backlog`：等待连接队列的最大长度（典型值5-10）

#### 5. `accept()` - 接受连接（接听电话）
**功能**：从监听队列中接受一个连接，返回新的连接Socket。
```c
#include <sys/socket.h>
int accept(int sockfd, struct sockaddr *addr, socklen_t *addrlen);
```
**参数**：
- `sockfd`：正在监听的Socket
- `addr`：用于存储客户端地址（可为NULL）
- `addrlen`：地址结构长度指针（可为NULL）

**重要**：返回的是**新的Socket描述符**，专门用于与这个客户端通信。

#### 6. `connect()` - 发起连接（拨打电话）
**功能**：客户端主动连接到服务器。
```c
#include <sys/socket.h>
int connect(int sockfd, const struct sockaddr *addr, socklen_t addrlen);
```
**参数**：
- `sockfd`：客户端的Socket
- `addr`：服务器地址信息
- `addrlen`：地址结构长度

### 三、数据收发函数

建立连接后，用这些函数发送接收数据。

#### 7. `write()` / `read()` - 流式数据传输
**功能**：像读写文件一样收发数据（用于已连接的流式Socket）。
```c
#include <unistd.h>
ssize_t write(int fd, const void *buf, size_t count);
ssize_t read(int fd, void *buf, size_t count);
```
**参数**：
- `fd`：连接建立的Socket描述符
- `buf`：数据缓冲区
- `count`：要读写的数据长度

#### 8. `send()` / `recv()` - 更专业的Socket数据传输
**功能**：功能类似read/write，但提供更多控制选项。
```c
#include <sys/socket.h>
ssize_t send(int sockfd, const void *buf, size_t len, int flags);
ssize_t recv(int sockfd, void *buf, size_t len, int flags);
```
**参数**：
- `flags`：控制标志，通常设为0
  - `MSG_OOB`：发送/接收带外数据（紧急数据）
  - `MSG_PEEK`：查看数据但不从缓冲区移除

#### 9. `sendto()` / `recvfrom()` - 无连接数据传输
**功能**：用于数据报Socket，每次发送都指定目标地址。
```c
#include <sys/socket.h>
ssize_t sendto(int sockfd, const void *buf, size_t len, int flags,
              const struct sockaddr *dest_addr, socklen_t addrlen);
              
ssize_t recvfrom(int sockfd, void *buf, size_t len, int flags,
                struct sockaddr *src_addr, socklen_t *addrlen);
```
**参数**：
- `dest_addr`：目标地址（sendto用）
- `src_addr`：来源地址（recvfrom用，可设为NULL）
- `addrlen`：地址结构长度

### 五、使用场景

#### 流式Socket（打电话模式）流程：
**服务器端**：`socket()` → `bind()` → `listen()` → `accept()` → `read()/write()` → `close()`
**客户端**：`socket()` → `connect()` → `read()/write()` → `close()`

#### 数据报Socket（发短信模式）流程：
**服务器端**：`socket()` → `bind()` → `recvfrom()` → `sendto()` → `close()`
**客户端**：`socket()` → `bind()`（可选）→ `sendto()` → `recvfrom()` → `close()`

#### 重要提醒：
1. **UNIX域特有**：地址使用文件路径，通信前确保路径可用，通信后记得`unlink()`删除socket文件
2. **错误处理**：所有函数失败都返回-1，记得检查返回值！
3. **头文件**：记得包含`<sys/socket.h>`, `<sys/un.h>`, `<unistd.h>`
